# Assets

This folder contains lightweight, text-based SVG placeholders used by the public `README.md`.

Replace them with real visuals when ready:
- `demo.svg` → ideally a short `demo.gif` (≤ 10MB) showing end-to-end usage
- `architecture.svg` → architecture diagram
- `benchmark_chart.svg` → benchmark chart/figure
- `logo.svg` → project logo (optional)

Guidelines:
- Keep assets small (GitHub-friendly).
- Do not commit runtime artifacts (benchmark JSON, smoke outputs).
